Implementación

En la etapa de implementación o codificación lo que se hace es traducir todas la información detallada en el diseño a código de máquina.

Para esta etapa es indispensable seleccionar las herramientas adecuadas, un entorno de desarrollo que facilite el trabajo y un lenguaje de programación apropiado según el tipo de sistema a desarrollar. Adicionalmente también se deben de tener en cuenta todos los recursos necesarios para que el sistema funcione, como son las licencias de uso de las bases de datos.

Al momento de la programación debemos de aplicar todas las técnicas requeridas para entregar un código legible como son:

  • Evitar estructuras de control no estructuradas.

  • Nombrar adecuadamente las variables.

  • Lógica sencilla

  • Documentar el código.

  • Utilizar la sangría.

  • Realizar código reutilizables.
